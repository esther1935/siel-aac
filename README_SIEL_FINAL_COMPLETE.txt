시엘 AAC 최종 완성 코딩 파일

버전: sielFinalComplete20260628

반영 기능:
1. 그림 관리
- 그림 추가
- 그림 수정
- 사진 교체
- 그림 삭제
- 그림 이름 수정
- 소리값은 그림 이름과 동일하게 자동 저장

2. 카테고리 관리
- 카테고리 추가
- 카테고리 이름 변경
- 카테고리 삭제
- 카테고리 순서 변경: ⬆️⬇️ 버튼 방식
- 드래그 순서 변경은 추후 안정화 단계에서 추가 권장

3. 백업
- 백업 다운로드 유지
- 백업 불러오기 유지

4. 오프라인
- sw.js 서비스워커 추가/유지
- manifest.webmanifest 추가/유지
- Wi-Fi 연결 상태에서 앱을 한 번 열면 앱 파일이 기기에 저장됨
- Wi-Fi 연결 상태에서 보인 그림은 기기 캐시에 저장됨
- 이후 Wi-Fi 없이도 마지막 저장된 그림과 데이터로 사용 가능
- Wi-Fi가 다시 연결되면 Firebase 동기화 재시도

5. Galaxy 미니탭 보정
- 휴대폰 화면 비율은 건드리지 않음
- 낮은 가로형 태블릿에서만 그림 영역을 키우고 글씨를 줄임
- 그림은 object-fit: contain으로 잘리지 않게 표시

적용 후 접속:
https://siel-aac.netlify.app/?v=sielFinalComplete20260628

중요:
새 그림을 추가한 뒤에는 시엘이 Galaxy Tab에서 Wi-Fi를 켠 상태로 앱을 한 번 열어 주세요.
그 후 Wi-Fi를 꺼도 마지막으로 받아온 그림으로 사용할 수 있습니다.
