시엘 AAC 깨끗한 최종본

이 ZIP은 GitHub에 전체 덮어쓰기용입니다.

포함된 핵심 파일:
- index.html
- app.js
- styles.css
- firebase-config.js
- firebase-config.sample.js
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png

적용 방법:
1. 이 ZIP 압축을 풉니다.
2. GitHub 저장소의 파일들을 이 파일들로 전체 덮어씁니다.
3. Commit changes를 누릅니다.
4. Netlify 배포가 Published 될 때까지 기다립니다.
5. 휴대폰에서 아래 주소로 접속합니다.
   https://siel-aac.netlify.app/?v=clean20260623

정상 상태 문구:
- 클라우드 데이터 반영 완료

이 문구가 뜨면 휴대폰도 Firestore 데이터를 읽고 있는 것입니다.
