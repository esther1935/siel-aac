시엘 AAC 관리자 + 카테고리 관리 + 오프라인 사용 버전

반영 내용:
1. 관리자 그림 관리
   - 그림 추가
   - 그림 수정
   - 사진 교체
   - 그림 삭제
   - 그림 이름 수정
   - 소리값은 그림 이름과 동일하게 자동 저장

2. 카테고리 관리
   - 카테고리 추가
   - 카테고리 이름 변경
   - 카테고리 삭제
   - ⬆️⬇️ 버튼으로 순서 변경
   - 드래그 순서 변경은 추후 안정화 단계에서 추가 권장

3. 백업
   - 백업 다운로드 유지
   - 백업 불러오기 유지

4. 오프라인
   - 서비스워커(sw.js) 추가
   - manifest.webmanifest 추가
   - Wi-Fi 연결 상태에서 앱을 한 번 열면 앱 파일이 기기에 저장됨
   - Wi-Fi 연결 상태에서 보인 그림은 기기 캐시에 저장되어 이후 오프라인에서도 표시됨
   - Firebase 연결이 안 되어도 localStorage에 저장된 마지막 데이터로 앱 실행

5. Galaxy 미니탭 보정
   - 휴대폰 화면은 건드리지 않고
   - 낮은 가로형 태블릿 화면에서만 글씨를 줄이고 그림 영역을 키움

적용 후 접속:
https://siel-aac.netlify.app/?v=adminOfflineCategory20260628

중요:
- 적용 후 한 번은 Wi-Fi를 켠 상태에서 앱을 열어 주세요.
- 새 그림을 추가한 뒤에도 해당 태블릿에서 Wi-Fi를 켜고 앱을 한 번 열어야 오프라인 저장됩니다.
