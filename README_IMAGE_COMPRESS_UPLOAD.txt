반영 내용:
- 관리자에서 그림 업로드 시 자동으로 이미지 용량을 줄입니다.
- 긴 변 기준 최대 1100px로 리사이즈합니다.

- 작은 원본보다 압축본이 더 커지는 경우 원본을 유지합니다.
- 업로드 후 동기화/오프라인 저장에 유리하도록 파일 크기를 줄입니다.
