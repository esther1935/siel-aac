시엘 AAC Responsive Fit 수정본

수정 핵심:
1. 노트북, Galaxy Tab, Z Fold/핸드폰 화면에서 모두 비율에 맞게 들어가도록 CSS를 다시 정리했습니다.
2. 고정 픽셀 중심 구조를 줄이고 clamp/vh/vw 기반 반응형 구조로 변경했습니다.
3. 문장 카드와 아래 선택 카드 이미지가 잘리지 않도록 이미지 박스를 다시 안정화했습니다.
4. 하단 카테고리가 화면 밖으로 밀리지 않도록 전체 레이아웃 높이를 재배분했습니다.
5. 아래 그림 선택 영역은 계속 YouTube처럼 좌우 스와이프됩니다.
6. 선택된 아래 카드는 #A6A6A6 테두리로 표시됩니다.
