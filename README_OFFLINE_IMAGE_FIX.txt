시엘 AAC 오프라인 이미지 깨짐 수정 버전

버전: sielOfflineImageFix20260628

수정 이유:
이전 버전은 Wi-Fi를 끄면 앱 화면과 데이터는 열렸지만 Firebase Storage 그림 요청이 서비스워커에서 이미지 캐시보다 먼저 제외되어 그림이 깨질 수 있었습니다.

수정 내용:
1. Firebase Storage 이미지도 오프라인 이미지 캐시에 저장
2. 앱 화면에 실제로 표시된 그림도 load 이벤트 기준으로 캐시 저장
3. 온라인에서 한 번 본 그림은 Wi-Fi 없이도 표시
4. 휴대폰/태블릿 화면 비율 기존 유지

적용 후 접속:
https://siel-aac.netlify.app/?v=sielOfflineImageFix20260628

중요:
적용 후에는 반드시 시엘이 탭에서 Wi-Fi를 켠 상태로 앱을 한 번 열고,
전체/각 카테고리를 한 번씩 눌러 그림을 화면에 보이게 해 주세요.
그 후 Wi-Fi를 끄고 다시 실행하면 그림이 표시됩니다.
