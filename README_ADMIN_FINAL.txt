시엘 AAC 관리자 최종 화면 수정본

반영 내용:
1. 관리자 화면을 보내주신 최종 시안 구조로 재구성했습니다.
2. 현재 그림 목록에 '수정' 버튼을 추가했습니다.
3. 수정 버튼을 누르면 기존 그림 이름과 카테고리가 입력칸에 올라옵니다.
4. 이름만 수정하거나, 새 사진을 선택해 사진 교체도 가능합니다.
5. 수정 저장 시 소리값은 그림 이름과 동일하게 자동 저장됩니다.
6. 삭제 기능과 백업 다운로드/불러오기는 유지했습니다.
7. 메인 AAC 최종 화면은 유지했습니다.
