시엘 AAC 관리자 최종 반영 버전

버전: sielAdminFinal20260628

반영 내용:
1. 세 줄/햄버거 버튼을 누르면 먼저 비밀번호 입력 화면이 뜹니다.
2. 비밀번호 통과 후 '그림 올리기' / '게시판' 메뉴가 보입니다.
3. 관리자 그림 목록이 너무 길어지지 않도록 12개씩 페이지로 나눕니다.
4. 새로 업로드하는 그림은 자동 압축되고, 그림 이름 옆에 흐린색으로 KB/MB 용량을 표시합니다.
5. 기존 그림은 용량 정보가 없는 경우 표시되지 않을 수 있으며, 수정/재업로드 이후 표시됩니다.
